title: "Quickstart"
description: "Create and run your first workflow in 5 minutes"
---

This guide walks you through building a working workflow — a Manual Trigger connected to an OpenAI node — so you can see AutoPilot in action as quickly as possible.

<Note>
  Creating workflows requires an active Pro subscription. If you are on the free tier, you will be prompted to upgrade when you click **New Workflow**. AI and notification nodes also require saved credentials — complete step 2 before adding them.
</Note>

<Steps>
  <Step title="Sign up or log in">
    Go to AutoPilot and create an account using your email address and a password. After signing up you are automatically signed in and redirected to the **Workflows** page.

    If you already have an account, click **Log in** and enter your credentials.
  </Step>

  <Step title="Add a credential">
    Before you can use an AI or notification node, you need to save the API key or token for that service.

    1. In the sidebar, click **Credentials**.
    2. Click **New Credential**.
    3. Give the credential a name (for example, `My OpenAI Key`).
    4. Select the **Type** — choose **OpenAI** for this walkthrough.
    5. Paste your OpenAI API key (starts with `sk-`) into the **API Key** field.
    6. Click **Create Credential**.

    The credential is now saved to your account and will appear in the node configuration panel when you add an OpenAI node.

    <Tip>
      You can add credentials for any supported service at any time — Anthropic, Gemini, Discord Webhook, Slack Webhook, Telegram Bot, and Email (SMTP) are all available.
    </Tip>
  </Step>

  <Step title="Create a new workflow">
    1. In the sidebar, click **Workflows**.
    2. Click **New Workflow**. AutoPilot creates an empty workflow and opens the visual canvas editor.

    The canvas shows a minimap in the lower right, zoom controls on the left, and an **Add node** button ( **+** ) in the upper right.
  </Step>

  <Step title="Add a Manual Trigger node">
    1. Click the **+** button in the upper right of the canvas.
    2. In the node selector, choose **Manual Trigger**.
    3. The node appears on the canvas. Drag it to position it on the left side.

    A Manual Trigger node starts the workflow when you click **Execute Workflow**. It does not require a credential. Once this node is on the canvas, the **Execute Workflow** button becomes visible at the bottom of the editor.
  </Step>

  <Step title="Add an OpenAI node and connect it">
    1. Click **+** again and choose **OpenAI**.
    2. The OpenAI node appears on the canvas.
    3. In the node's configuration panel, select the credential you saved in step 2 from the **Credential** dropdown.
    4. Enter the prompt you want to send to the model.
    5. Draw a connection from the **Manual Trigger** node's output handle to the **OpenAI** node's input handle by clicking and dragging between the two handles.

    The connection tells AutoPilot to run the OpenAI node after the trigger fires, passing any context data along.
  </Step>

  <Step title="Execute the workflow and view results">
    1. Click **Execute Workflow** at the bottom of the canvas.
    2. AutoPilot sends the workflow to the background job runner and executes each node in order.
    3. Results stream back in real time — each node on the canvas updates its status indicator as it completes (loading → success or error).
  </Step>
</Steps>

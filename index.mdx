## What you can build

AutoPilot workflows are composed of nodes connected together on a visual canvas. Each run executes nodes in the correct order, passing data between them automatically.

## Get started

<Steps>
  <Step title="Create an account">
    Sign up at AutoPilot and log in to your dashboard.
  </Step>
  <Step title="Add credentials">
    Go to **Credentials** and add API keys for the services you want to use — OpenAI, Anthropic, Slack, and more.
  </Step>
  <Step title="Build a workflow">
    Open the **Workflows** section, click **New Workflow**, and drag nodes onto the canvas to build your automation.
  </Step>
  <Step title="Run it">
    Click **Execute** to run your workflow and watch results stream in real time.
  </Step>
</Steps>

## Supported node types

AutoPilot supports 11 node types spanning triggers, AI models, and notification services:

| Category | Nodes |
|---|---|
| **Triggers** | Manual, HTTP Request, Google Form, Stripe |
| **AI Models** | OpenAI, Anthropic (Claude), Google Gemini |
| **Notifications** | Discord, Slack, Telegram, Email (SMTP) |

<Note>
  All credentials are stored securely and scoped to your account. API keys are never exposed to the workflow canvas.
</Note>
